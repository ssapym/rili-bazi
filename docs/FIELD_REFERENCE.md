# 八字排盘系统字段参考手册

本文档详细说明 API 返回的所有 JSON 字段含义及数据结构。

## 命名规范

### 通用规则

- 通用英文术语使用英文，如 baseInfo、age、gender
- 命理专用术语使用拼音，如 tiangan、dizhi、nayin、shensha
- 组合概念也使用拼音，如 canggan、fuxing、xingyun

### 第一层模块

| 字段 | 类型 | 说明 |
|------|------|------|
| baseInfo | object | 基本信息，包含用户出生基本数据 |
| sizhu | object | 四柱信息，年、月、日、时 |
| chonghe | object | 刑冲合害关系 |
| nengliang | object | 能量分析 |
| geju | object | 格局分析 |
| dayun | object | 大运流年 |
| shensha | array | 神煞数组（包含在 sizhu 中） |

## baseInfo（基本信息）

### 数据结构

```json
{
  "baseInfo": {
    "solarBirthday": "1997年4月11日 10:00",
    "lunarBirthday": "丁丑三月初五丁巳时",
    "gender": "女性",
    "chineseZodiac": "丑 (牛)",
    "zodiac": "白羊座",
    "age": 28,
    "bazi": "丁丑 甲辰 癸未 丁巳",
    "baziArr": {
      "niangan": "丁",
      "nianzhi": "丑",
      "yuegan": "甲",
      "yuezhi": "辰",
      "rigan": "癸",
      "rizhi": "未",
      "shigan": "丁",
      "shizhi": "巳"
    }
  }
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| solarBirthday | string | 公历生日，格式如「1997年4月11日 10:00」 |
| lunarBirthday | string | 农历生日，格式如「丁丑三月初五丁巳时」 |
| gender | string | 性别，「男性」或「女性」 |
| chineseZodiac | string | 生肖，格式如「丑 (牛)」 |
| zodiac | string | 星座，格式如「白羊座」 |
| age | number | 虚岁年龄 |
| bazi | string | 八字字符串，格式如「丁丑 甲辰 癸未 丁巳」 |
| baziArr | object | 八字数组对象 |

### baziArr 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| niangan | string | 年干 |
| nianzhi | string | 年支 |
| yuegan | string | 月干 |
| yuezhi | string | 月支 |
| rigan | string | 日干 |
| rizhi | string | 日支 |
| shigan | string | 时干 |
| shizhi | string | 时支 |

## chonghe（刑冲合害关系）

### 数据结构

```json
{
  "chonghe": {
    "stems": [
      { "source": "年+月", "desc": "甲己合化土", "type": "合" }
    ],
    "branches": [
      { "source": "年+月", "desc": "子丑合化土", "type": "合" }
    ],
    "pillars": [
      { 
        "source": "年+月", 
        "desc": "甲子合己丑", 
        "type": "双合",
        "details": "甲子与己丑"
      }
    ]
  }
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| stems | array | 天干关系数组 |
| branches | array | 地支关系数组 |
| pillars | array | 干支组合关系数组 |

### 关系对象

| 字段 | 类型 | 说明 |
|------|------|------|
| source | string | 关系源柱位，如「年+月」 |
| desc | string | 关系描述，如「甲己合」 |
| type | string | 关系类型：天干五合、天干相克、地支三合、地支六合、六冲、相害、六破、拱会、双冲、天克地刑、双合 |
| details | string | 详细说明（仅pillars字段），如「甲子与己丑」 |

### 关系类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| 天干五合 | 两干相合，化气生财 | 甲己合化土、乙庚合化金 |
| 天干相克 | 五行相克关系，克制对方 | 甲克戊、丙克庚 |
| 地支三合 | 三支合局，力量强大 | 申子辰合水、亥卯未合木 |
| 地支六合 | 两支相合，关系密切 | 子丑合土、寅亥合木 |
| 六冲 | 两支相冲，冲突对立 | 子午冲、丑未冲 |
| 相害 | 两支相害，暗中损害 | 子未害、丑午害 |
| 六破 | 两支相破，破坏关系 | 子酉破、卯午破 |
| 拱会 | 两支夹合，力量次之 | 申巳拱会午 |
| 双冲 | 天干相冲且地支也相冲 | 甲子冲庚午、乙丑冲辛未 |
| 天克地刑 | 天干相克且地支自刑 | 乙亥克己亥、戊辰克壬辰 |
| 双合 | 天干相合且地支也相合 | 甲子合己丑、丙寅合辛亥 |

### 天干相克关系

天干相克遵循五行相克原理：木克土、火克金、土克水、金克木、水克火。

| 相克关系 | 五行原理 | 说明 |
|----------|----------|------|
| 甲克戊 | 木克土 | 甲木克制戊土 |
| 乙克己 | 木克土 | 乙木克制己土 |
| 丙克庚 | 火克金 | 丙火克制庚金 |
| 丁克辛 | 火克金 | 丁火克制辛金 |
| 戊克壬 | 土克水 | 戊土克制壬水 |
| 己克癸 | 土克水 | 己土克制癸水 |

### 地支六破关系

地支六破是指地支之间存在的破坏关系，通常表示关系的破裂或破坏。

| 六破关系 | 说明 |
|----------|------|
| 子酉破 | 子水与酉金相破，金水相破 |
| 卯午破 | 卯木与午火相破，木火相破 |
| 辰丑破 | 辰土与丑土相破，土土相破 |
| 未戌破 | 未土与戌土相破，土土相破 |
| 寅亥破 | 寅木与亥水相破，木水相破 |
| 申巳破 | 申金与巳火相破，金火相破 |

### 干支组合关系

干支组合关系是指两个完整的干支柱之间同时存在天干关系和地支关系的情况。
